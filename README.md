# peduha2.0

Flask-сайт коледжу з адмінкою, SQLite-базою та єдиною структурою фронтенду.

## Структура

- `app.py` - головний Flask-застосунок і маршрути.
- `templates/` - усі Jinja-шаблони, які рендерить Flask.
- `static/` - єдине джерело фронтенд-асетів:
  - `static/css/` - стилі
  - `static/js/` - клієнтський JavaScript
  - `static/images/` - зображення інтерфейсу
  - `static/uploads/` - завантажені через адмінку файли
- `pages/` - HTML-вміст для сторінок `/page/<slug>`.
- `data/` - локальна SQLite-база.
- `legacy/` - архів старих standalone HTML-файлів, які не використовуються Flask напряму.

## Поточний принцип

- Усі активні сторінки сайту працюють через `templates/` + `static/`.
- Дублікати кореневих `css/`, `js/` та `images/` більше не використовуються.
- Старі standalone сторінки винесені в `legacy/` для історії.
